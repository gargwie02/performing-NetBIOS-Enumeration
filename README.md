# performing-NetBIOS-Enumeration
## SMB Enumeration Lab

### Objective

To gather information about shared resources and users using SMB enumeration techniques.

### Tools Used

* Nmap
* smbclient
* enum4linux

### Procedure

1. Performed Nmap scan to check SMB port (445)
2. Used smbclient to list shared folders
3. Used enum4linux to enumerate users and system information

### Result

Successfully identified:

* Shared folders
* Users and groups
* System information

### Conclusion

SMB enumeration helps in extracting sensitive information from target systems, which is useful in penetration testing.
